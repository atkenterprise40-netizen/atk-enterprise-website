# ATK Enterprise (Pvt) Ltd — Light Theme Website

This is a simple, multi-page, light-themed website for ATK Enterprise (Pvt) Ltd.

Included files:
- index.html (Home)
- about.html
- services.html
- contact.html
- style.css

How to use:
1. Upload all files to your GitHub repo (root level).
2. Enable GitHub Pages from Settings → Pages → Deploy from a branch → main / (root).
3. Wait a few minutes and open your site at: https://<your-username>.github.io/atk-enterprise-website/

Replace the email in contact.html when you have your domain email ready.
